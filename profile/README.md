## Mission

**OI Lab** is an open-source initiative driven by a single conviction: great developer tooling should be **invisible** — it solves real problems without friction, and disappears into your workflow.

Every package we ship is born from production constraints, real codebases, and the relentless pursuit of developer experience. We build at the intersection of **Laravel**, **TypeScript**, and **AI-augmented development** — tools that bridge the gap between elegant backend architecture and modern frontend expectations.

## Open Source Projects

### Laravel Packages

| Package | Description |
|---|---|
| [**oi-laravel-ts**](https://github.com/oi-lab/oi-laravel-ts) | Automatically generates TypeScript interfaces from your Eloquent models — with relationships, custom casts, and DataObjects support. |
| [**oi-laravel-documentation**](https://github.com/oi-lab/oi-laravel-documentation) | Manages markdown-based documentation with hierarchical navigation and full-text search, directly within your Laravel app. |

> More packages are in the pipeline. Watch this space. ⭐
